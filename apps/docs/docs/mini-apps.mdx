# Mini-Apps

## Welcome to Niwi Starter Mini Apps!

Niwi Starter is packed with **mini-apps** designed to help you level up your React.js skills. Whether you’re a beginner or looking for fun projects, we’ve got you covered! 🚀

### Explore What You Can Build:

- ✅ **To-Do List App:** Manage your tasks efficiently with this fully functional app.
- ⏲️ **Pomodoro Timer:** Boost productivity with the classic time management technique.
- 🌐 **Preview Screen:** Enter any website URL and see a screenshot preview.
- ❤️ **Wallet App:** Track your expenses and manage your budget.

### 🎯 Getting Started is Easy!

No matter your experience level, getting started is a breeze. Just run one of the commands below:

<Tabs>
  <TabItem value="npx" label="NPX">
    ``` npx create-niwi ```
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    ``` yarn create niwi ```
  </TabItem>
  <TabItem value="pnpm" label="PNPM">
    ``` pnpm create niwi ```
  </TabItem>
</Tabs>

### 📂 Dive Into the Mini-Apps Directory

You can check out and learn from all the mini-apps in the directory:  

`app/(feat)/mini-app`.

Explore the code, tweak it, and see how each app works under the hood!
