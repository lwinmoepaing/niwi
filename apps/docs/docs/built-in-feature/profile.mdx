# Niwi Profile

## Profile Feature Overview

<div className="relative rounded-[12px] overflow-hidden my-5">
  <img className="w-full" src="/profile/profile.gif" />
</div>


Easily showcase and organize your linked cards—such as Facebook, Twitter, YouTube channel, GitHub, and Buy Me a Coffee—into a collection of profile cards that you can share.

With intuitive drag-and-drop functionality, you can quickly arrange and reposition these cards to create a personalized profile that stands out.

Once you’ve set it up to your liking, sharing your profile is just a click away.
